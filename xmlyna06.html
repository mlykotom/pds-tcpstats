About

Script for parsing input PCAP file and outputting results as JSON.
Program parses:

  • Window sizes
  • Transfer speed
  • Roundtrip time
  • Sequence analyse

Usage

Run script with either command:

  • ./tcpstats INPUT_FILE [OUTPUT_FILE] (note: executing rights must be set)
  • python tcpstats INPUT_FILE [OUTPUT_FILE]

If OUTPUT_FILE not specified, default ./log/log.json file is used.

Used technologies

  • JetBrains PyCharm
  • Python 2.7
  • DPKT library
  • XHTML
  • Bootstrap
  • jQuery
  • HighCharts

Notes

  • Might be needed to install DPKT library.
    make install
  • Google Chrome prevents opening local files, so it won't open log file. Tested browsers are Mozilla Firefox, Safari.
  • Other way to load data is to insert log into this HTML page as javascript and add to log file initializing variable.
