About

Script for parsing input PCAP file and outputting results as JSON.
Program parses:

  • Window sizes
  • Transfer speed
  • Roundtrip time
  • Sequence analyse

Usage

Run script with either command:

  • ./tcpstats INPUT_FILE [OUTPUT_FILE]
  • python tcpstats INPUT_FILE [OUTPUT_FILE]

If OUTPUT_FILE not specified, default ./log/log.json file is used.

  • Python 2.7
  • DPKT library

  • Bootstrap
  • jQuery
